# Student-Result-Processing-System
# Project Report (1–2 Pages)
# Introduction

The Student Result Processing System is a database application developed using SQL Server. It manages student information, course details, grades, GPA calculations, and ranking reports.

# Abstract

The project automates student result processing and generates academic reports. SQL queries, views, triggers, and ranking functions are used to calculate GPA and analyze student performance.

# Tools Used

SQL Server
SSMS (SQL Server Management Studio)

# Steps Involved

Created StudentResultDB database.
Created Students, Courses, Semesters, and Grades tables.
Inserted sample data.
Implemented GPA calculation.
Created rank list using Window Functions.
Created View and Trigger.
Generated reports.

# Conclusion

The project successfully manages student academic records and automates result generation, GPA calculation, and ranking analysis using SQL Server.
